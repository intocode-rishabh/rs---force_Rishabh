🌟 Project Overview

The Neighbourhood Service Marketplace is a full-stack web platform designed to connect local service providers (plumbers, tutors, electricians, delivery agents, doctors, etc.) with customers in their neighborhood.

The goal is to simplify how people find reliable services while also creating better earning opportunities for local workers through a trusted, organized, and interactive system.

❗ Problem Statement

Finding trustworthy local service providers is often:

Time-consuming
Unreliable
Unorganized

Customers struggle to compare options, while service providers lack visibility and consistent work opportunities.

💡 Our Solution

We built a platform where:

Customers can find services or post requests
Service providers can find work based on their skills
Users can connect directly through chat
Profiles help build trust and transparency
AI enhances user bios for better presentation
🔑 Key Features
🔐 Authentication System (Customer & Service Provider roles)
👤 Profile Creation & Management
🔍 Find Service / Find Work Modules
📝 Post Service Requests
💬 One-to-One Chat System with History
🤖 AI Bio Enhancer (Grok API Integration)

📱 Responsive & Clean UI

⚙️ How to Run Locally
1️⃣ Clone the repository
git clone <your-repo-link>
cd NeighbourHub
2️⃣ Install dependencies
npm install
3️⃣ Setup environment variables

Create a .env file:

MONGO_URI=your_mongodb_connection_string
PORT=5000
4️⃣ Start the server
npm start
5️⃣ Open in browser
http://localhost:5000
