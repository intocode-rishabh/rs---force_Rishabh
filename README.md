## 🌟 Project Overview
The **Neighbourhood Service Marketplace** is a full-stack web platform that connects **local service providers** (plumbers, tutors, electricians, delivery agents, doctors, etc.) with **customers in their neighborhood**.

🎯 The goal is to:
- Simplify service discovery  
- Improve trust and transparency  
- Create earning opportunities for local workers  

---

## ❗ Problem Statement
Finding reliable local services is often:
- ⏳ Time-consuming  
- ❌ Unreliable  
- 📉 Unorganized  

Customers struggle to compare options, while service providers lack visibility and consistent work opportunities.

---

## 💡 Our Solution
We built a platform where:
- 🔍 Customers can find services or post requests  
- 🛠️ Service providers can find work based on skills  
- 💬 Users can connect via real-time chat  
- 👤 Profiles build trust and transparency  
- 🤖 AI enhances user bios for better presentation  

---

## 🔑 Key Features
- 🔐 **Authentication System** (Customer & Service Provider roles)  
- 👤 **Profile Creation & Management**  
- 🔍 **Find Service / Find Work Modules**  
- 📝 **Post Service Requests**  
- 💬 **One-to-One Chat System with History**  
- 🤖 **AI Bio Enhancer (Grok API Integration)**  
- 🧪 **Dummy Data for Demo Experience**  
- 📱 **Responsive & Clean UI**
